TAGS HTML

Aqui iremos ver um pouco sobre algumas das tags existentes na linguagem de marcação HTML

  • h1 a h6: as tags <h1> a <h6> em HTML são usadas para criar títulos ou cabeçalhos em uma página da web. Existem seis níveis diferentes de cabeçalhos, onde <h1> é o mais importante e <h6> é o menos importante. Aqui está uma explicação simples de cada uma delas

    • exemplos


  • <p>A tag <p> em HTML é usada para criar parágrafos de texto. Ela é uma das tags mais simples e fundamentais em HTML. Quando você envolve um pedaço de texto com a tag <p>, o navegador exibe esse texto em um parágrafo separado, com espaço em branco acima e abaixo dele, criando assim uma separação visual entre os parágrafos.
    • exemplos


  • <i>A tag em HTML é usada para aplicar estilo de texto em itálico a uma parte do conteúdo. Quando você envolve um texto com a tag <i>, o navegador normalmente o exibirá em itálico, o que significa que o texto será inclinado para a direita para enfatizar ou destacar essa parte do conteúdo.
    • exemplos


  • <mark>A tag <mark> é usada em HTML para destacar ou realçar um pedaço de texto dentro de um parágrafo ou outro elemento de texto. Quando você envolve um texto com a tag <mark>, o navegador geralmente o apresenta de forma a chamar a atenção do leitor, tornando-o geralmente com um fundo amarelo ou outra cor de destaque, dependendo do estilo do navegador.
    • exemplos


  • <small> tag <mark> em HTML é usada para indicar que o texto dentro dela deve ser exibido em um tamanho de fonte menor do que o texto circundante. Isso é útil quando você deseja incluir informações menos importantes ou notas de rodapé em um tamanho de fonte menor para distingui-las do texto principal.
    • exemplos


  • <u>A tag <u> em HTML é usada para sublinhar o texto que a envolve. Quando você coloca texto entre <u> e <u>, o navegador exibe esse texto com uma linha sublinhada, como se fosse um link, mas sem ser um link clicável.
    • exemplos


  • <sub>A tag <sub> em HTML é usada para criar texto subscrito, o que significa que o texto dentro dessa tag será exibido em um nível mais baixo em relação ao texto circundante, como em fórmulas químicas ou equações matemáticas, onde os números inferiores são frequentemente usados.
    • exemplos


  • <sup>A tag <sup> em HTML é usada para criar texto sobrescrito, o que significa que o texto dentro dessa tag será exibido em um nível mais alto em relação ao texto circundante. Isso é comumente usado em fórmulas matemáticas, unidades de medida, notas de rodapé e outras situações onde é necessário mostrar texto ligeiramente acima do nível padrão.
    • exemplos


  • <blockquote>A tag <blockquote> em HTML é usada para citar um bloco de texto que foi retirado de outra fonte, como um livro, artigo, site ou qualquer outra fonte escrita. Ela é geralmente usada para destacar e separar visualmente uma citação do restante do conteúdo. Quando você envolve um texto com a tag <blockquote>, geralmente ele é exibido com recuo nas margens esquerda e direita, indicando que se trata de uma citação.
    • exemplos


  • <del>A tag <del> em HTML é usada para marcar o texto que foi removido ou deletado de uma versão anterior de um documento. Quando você envolve um texto com a tag <del>, o navegador normalmente exibe esse texto com uma linha através dele ou em um estilo de texto diferente para indicar que foi removido.
    • exemplos


  • <abbr>A tag <abbr> em HTML é usada para definir uma abreviação ou uma sigla e fornecer uma explicação ou descrição completa para essa abreviação. Quando você usa a tag <abbr>, o navegador normalmente exibe a abreviação com destaque, e quando o usuário passa o mouse sobre ela, uma dica de ferramenta (tooltip) é exibida com a descrição completa.
    • exemplos


  • <Strong> A tag <Strong> em HTML é usada para enfatizar o texto, indicando que o texto dentro dela deve ser exibido em negrito. É uma forma de dar ênfase ao conteúdo para torná-lo mais destacado e significativo no contexto do restante do texto.
    • exemplos


  • <hr>A tag <hr> em HTML é usada para criar uma linha horizontal, que geralmente é usada para separar o conteúdo em seções distintas de uma página ou para marcar uma divisão visual entre elementos. É frequentemente utilizada para criar uma linha divisória horizontal no layout de uma página da web.
    • exemplos
